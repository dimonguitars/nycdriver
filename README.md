# nycdriver
Mobile app for NYC drivers
Allows cstmr to track and pay parking tickets based on provided plate number. 

I used NYC open database API for all tickets details. 

Tech stack: 
React-Native + Redux;
Firebase Auth and DB;
Rest API;
React-native-elements;
